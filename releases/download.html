SRS Binary

推荐使用docker，在docker直接运行， 或在docker编译和调试， 若需要打SRS的安装包，可以运行下面的命令打包： 

echo "Package SRS 3.0 for CentOS7" &&
git clone https://github.com/ossrs/srs.git && cd srs && git checkout 3.0release &&
docker run --rm -it -v `pwd`:/srs -w /srs/trunk registry.cn-hangzhou.aliyuncs.com/ossrs/srs:dev \
    ./scripts/package.sh --x86-x64

SRS Usage

echo "For CentOS7" &&
unzip -q SRS-CentOS7-x86_64-*.zip && cd SRS-CentOS7-x86_64-* && sudo bash INSTALL &&
sudo systemctl start srs
